## Build

You can build the project in two different ways.

#### Node
```
# navigate to the nextjs project folder
cd frontend/

# install all necessary dependencies
yarn install

# build the frontend
yarn build

# start the frontend (default port: 3000)
yarn start 
```

#### Docker (Compose)

```
# Start (exposed port: 3000)
docker-compose up -d
```
